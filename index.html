About This Website



This website is about web design created by Kyle Suess as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2023. All the text comes from the following articles located on the Dribbble website. All the images were created by me using Photoshop. The website has been created using HTML and CSS in Dreamweaver.

  • I. Dribbble: Choosing typography for web design by Renee Fleck
  • II. Dribbble: Choosing colors for web design by Renee Fleck
  • III. Dribbble: 11 Principles of Design by Renee Fleck

About Kyle:

Kyle Suess is a graphic design major currently studying at North Central Colllege. In addition to studying graphic design, he also works as a tutor at the college's writing center, helping students to better their writing skills.
